# Changelog

## v0.3.2
- Added more save options ( [s] save, [q] quit, [w] save and quit )
- Improved status display and prompts
- Files are now only created when saved
  
## v0.3.1
- Fixed buffer handling bug that caused unintended characters to be saved

## v0.3.0
- Added cursor support with left/right arrow key
- Improved visual cursor rendering
- Improved in-buffer insertion and deletion
  
## v0.2.0
- Initial usable version
- Rewritten to use an in-memory buffer for editing
- Interactive editor loop
- Basic backspace and newline handling
- Normal / Insert modes (vim-inspired)
- Create new files or open existing ones
- Save edited contents on exit
- Status display showing file name, mode, and position

## v0.1.0
- Proof-of-concept
- Open and display contents of an existing file
- Append text to the file
