# Changelog

## v0.2.0
- Initial usable version
- Rewritten to use an in-memory buffer for editing
- Interactive editor loop
- Basic backspace and newline handling
- Normal / Insert modes (vim-inspired)
- Create new files or open existing ones
- Save edited contents on exit
- Status display showing file name, mode, and position

## v0.1.0
- Proof-of-concept
- Open and display contents of an existing file
- Append text to the file
